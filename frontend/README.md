Interface - Acervo de Filmes (Frontend)

Este é o frontend do projeto final, desenvolvido em React. Ele permite visualizar, cadastrar, editar e excluir filmes do catálogo.

Tecnologias Utilizadas
- React (Create React App)
- React Router Dom (Navegação)
- Axios (Requisições HTTP)
- CSS Modules (Estilizaçao)

Funcionalidades
- Página Inicial: Lista todos os filmes.
- Cadastro: Formulário para novos filmes.
- Edição: Permite alterar dados.
- Exclusão: Remove filmes da lista.

Como Rodar o Projeto

1. Olhe se o Backend já está rodando.
2. Abra um terminal na pasta "frontend".
3. Instale as dependências:
   npm install
4. Execute o projeto:
   npm start
5. O navegador abrira em http://localhost:3000
